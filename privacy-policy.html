1. 引言

欢迎使用 OpenClaw（以下简称"我们"或"本服务"）。我们非常重视您的隐私保护，本隐私政策旨在向您说明我们如何收集、使用、存储、共享和保护您的个人信息，以及您享有的相关权利。

💡 重要提示：
• OpenClaw 是一个本地运行的 AI 助手框架，大部分数据处理在您的设备上完成
• 我们不会主动收集您的个人聊天记录和文件内容
• 请仔细阅读本政策，以了解我们对您信息的处理方式

通过使用我们的服务，您同意按照本隐私政策的规定处理您的信息。如果您不同意本政策的任何内容，请立即停止使用我们的服务。

2. 信息收集

2.1 我们收集的信息类型

📋 您主动提供的信息

  • 账户信息： 当您配置 AI 模型时，可能需要提供 API 密钥、OAuth 令牌等认证信息（这些信息存储在本地）
  • 配置信息： 您的偏好设置、工作区路径、渠道配置等
  • 反馈信息： 当您向我们提交问题报告或建议时提供的信息

🤖 自动收集的信息

  • 设备信息： 操作系统类型、版本号、设备型号
  • 使用日志： 启动时间、会话时长、功能使用频率（匿名统计）
  • 错误日志： 崩溃报告、错误堆栈（用于问题排查）

📝 本地存储的信息

🏠 本地数据（不上传到我们的服务器）：
• 聊天记录和对话历史
• 工作区文件和文档
• 记忆文件（MEMORY.md、日常笔记）
• 自定义技能和配置

2.2 我们不收集的信息

  • ❌ 您的真实姓名、身份证号、电话号码（除非您主动配置）
  • ❌ 您的聊天内容和个人文件（仅存储在本地）
  • ❌ 您的银行账户或支付信息（我们不直接处理支付）
  • ❌ 精确的地理位置信息

3. 信息使用

3.1 我们如何使用收集的信息

  • 提供服务： 运行 AI 助手、处理您的请求、保存配置
  • 改进产品： 分析使用模式，优化功能和性能
  • 技术支持： 诊断和解决技术问题
  • 安全保护： 检测和防止欺诈、滥用或非法活动
  • 通信联系： 发送重要通知、更新公告（需您同意）

3.2 信息共享

⚠️ 请注意： OpenClaw 本身不会共享您的个人信息，但以下情况除外：
  • 第三方 AI 模型： 当您使用云端 AI 模型（如 Kimi、DeepSeek、Claude 等）时，您的对话内容会发送到对应服务商的服务器，受其隐私政策约束
  • 法律要求： 根据法律法规要求、诉讼需要或政府机关要求
  • 业务转让： 如发生合并、收购或资产转让，我们会要求新的持有方继续受本政策约束
  • 您明确同意： 在您明确授权的情况下与其他方共享

3.3 数据保留

  • 本地数据： 只要您安装并使用 OpenClaw，数据会保留在您的设备上
  • 日志文件： 错误日志和诊断信息默认保留 90 天
  • 删除数据： 卸载 OpenClaw 时，您可以选择删除所有本地数据

4. Cookie 和追踪技术

4.1 什么是 Cookie

Cookie 是网站存储在您设备上的小型文本文件，用于记录您的偏好和浏览活动。

4.2 我们使用的 Cookie 类型

类型 用途 保留时间
必要 Cookie 网站基本功能运行（如会话管理） 会话期间
功能 Cookie 记住您的偏好设置（如深色模式） 1 年
分析 Cookie Google Analytics 统计访问量（可选） 2 年
广告 Cookie Google AdSense 展示相关广告 90 天

4.3 第三方 Cookie

我们的网站可能包含第三方服务的 Cookie，例如：

  • Google AdSense： 用于展示个性化广告
  • Google Analytics： 用于网站流量分析
  • GitHub Pages： 如果您通过 GitHub Pages 访问

4.4 管理 Cookie

您可以通过浏览器设置管理或禁用 Cookie：

  • Chrome： 设置 → 隐私和安全 → Cookie 及其他网站数据
  • Firefox： 选项 → 隐私与安全 → Cookie 和网站数据
  • Safari： 偏好设置 → 隐私 → 管理网站数据
  • Edge： 设置 → Cookie 和网站权限
⚠️ 注意： 禁用某些 Cookie 可能会影响网站功能

6. 数据安全

6.1 安全措施

我们采取合理的技术和组织措施保护您的信息安全：

  • 本地加密： 敏感配置信息（如 API 密钥）可加密存储
  • 访问控制： 工作区文件权限管理
  • 安全更新： 定期发布安全补丁和版本更新
  • 代码审计： 开源代码接受社区监督

6.2 您的安全责任

⚠️ 请注意： 由于 OpenClaw 主要在您的本地设备上运行，您也负有重要的安全责任：
  • 妥善保管您的 API 密钥和认证信息
  • 定期更新 OpenClaw 到最新版本
  • 不要将配置文件提交到公共代码仓库
  • 使用强密码保护您的设备
  • 定期备份重要数据

6.3 数据泄露通知

如果我们发现任何可能影响您信息安全的数据泄露事件，我们将在 72 小时内通过可用渠道通知您。

7. 您的权利

7.1 访问权

您有权访问我们持有的您的个人信息。您可以随时查看本地配置文件和日志。

7.2 更正权

您可以随时修改您的配置信息、偏好设置等。

7.3 删除权

您有权要求删除您的数据：

  • 删除特定数据： 手动删除配置文件或日志
  • 完全卸载： 运行 openclaw uninstall --all 删除所有数据

7.4 导出权

您可以导出您的数据：

# 导出聊天记录
openclaw sessions export --format json --output backup.json

# 备份工作区
cp -r ~/.openclaw/workspace /your/backup/location

7.5 撤回同意

您可以随时撤回对数据处理的同意，但这可能影响某些功能的正常使用。

7.6 投诉权

如您对我们的隐私处理有异议，您有权向相关监管机构投诉。

8. 儿童隐私

我们的服务不面向 13 岁以下的儿童。我们不会故意收集 13 岁以下儿童的个人信息。

👶 家长须知：
• 如果您发现您的孩子向我们提供了个人信息，请联系我们立即删除
• 建议家长监督未成年人的网络活动
• 教育儿童不要在网上分享个人信息

重大变更时，我们会通过以下方式通知您：

📅 建议： 定期查看本页面，了解最新的隐私政策
